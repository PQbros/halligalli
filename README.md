# halligalli [(developed version)](https://github.com/PQbros/halligalli-cmd-line-version)
### 服务端环境配置 (Python Flask)
* pipenv
    ```shell script
    # 安装pipenv
    $ pip install pipenv
    
    # 进入虚拟环境
    $ pipenv shell
    
    # 安装依赖
    $ pipenv install
    
    # 启动服务
    $ python3 server.py
    ```
---
### 客户端环境配置 (VUE)
* NodeJS
    ```shell script
    # Project setup
    npm install
    
    # Compiles and hot-reloads for development
    npm run serve
    
    # Compiles and minifies for production
    npm run build
    
    # Lints and fixes files
    npm run lint
    ```
